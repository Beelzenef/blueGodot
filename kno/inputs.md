# _Inputs_

Pulsando teclas:

```py
if Input.is_action_pressed("ui_left"):
	// Acción de pulsar a la izquierda
```

Obtener posición absoluta del ratón, para utilizarla con un nodo específico:

```py
var posMouse = get_global_mouse_pos()
set_pos(posMouse)
```
